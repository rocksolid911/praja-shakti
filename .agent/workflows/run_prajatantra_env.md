---
description: Run commands in the prajatantra conda environment
---

Follow these steps when you need to run terminal commands within the `prajatantra` conda environment:

1. Open the terminal and activate the `prajatantra` conda environment by running the following command:
   ```yaml
   conda activate prajatantra
   ```

2. After the environment is successfully activated, proceed with executing the required commands for the backend or scripts.
